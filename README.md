# 💫 About Me:
📱✨ SoulPass<br>O SoulPass é uma solução que transforma pontos da plataforma SoulUp 🎯 em créditos para transporte público 🚍. A proposta é incentivar ações sustentáveis 🌱, permitindo que usuários usem seus pontos para pagar viagens de forma prática ✅ e segura 🔒.<br>Além de facilitar o acesso à mobilidade urbana 🏙️, o SoulPass promove impacto ambiental positivo 🌍, conectando o engajamento digital 💻 a benefícios reais no dia a dia.
